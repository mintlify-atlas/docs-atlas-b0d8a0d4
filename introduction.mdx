LLM Internals is a structured learning resource that takes you from the first principles of how a large language model processes text all the way through to the techniques that make inference fast at scale. Built by [Amit Shekhar](https://x.com/amitiitbhu), Founder of Outcome School, the resource is designed for engineers, developers, and ML practitioners who want to move beyond using LLMs as black boxes and understand exactly what is happening inside them.

## What this resource covers

Each section focuses on one layer of the stack — starting with how raw text becomes tokens, building up through the attention mechanism and transformer architecture, and finishing with the inference optimization techniques that power production LLM deployments.

## How to use this resource

You can work through the topics in order for a complete picture of how LLMs work, or jump directly to the section most relevant to what you are building or learning right now.

  If you are new to LLMs, start with the [What Is an LLM?](/what-is-llm) page to get a grounding in the key concepts before diving into internals.
